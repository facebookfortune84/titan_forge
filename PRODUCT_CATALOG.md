# Product Catalog & Pricing Plans

This document outlines the core offerings for TitanForge, covering both AI-as-a-Service (AIaaS) platform subscriptions and productized Agentic Software Development Services. All prices are in USD.

## 1. AI-as-a-Service (AIaaS) Platform Subscriptions

Our AIaaS platform provides self-serve access to the TitanForge agent swarm for various development and operational tasks.

### 1.1. Starter Tier
*   **Target User Profile:** Individual developers, small startups, students, or users experimenting with AI agents for personal projects.
*   **Key Features:**
    *   Access to core agent swarm capabilities (e.g., basic code generation, simple task orchestration).
    *   Limited concurrent agent runs (1 concurrent task).
    *   Up to 1,000 API requests per month.
    *   Standard email support (48-hour response time).
    *   Access to public community forums.
    *   Basic task dashboard for monitoring agent activity.
*   **Price:**
    *   **Monthly:** $19 USD
    *   **Annually:** $199 USD (approximately 15% discount)
*   **Billing Model:** Monthly or Annual subscription.

### 1.2. Pro Tier
*   **Target User Profile:** Small to medium-sized businesses (SMBs), growing development teams, professional freelancers needing more robust agent capabilities.
*   **Key Features:**
    *   **All Starter Tier features.**
    *   Expanded concurrent agent runs (up to 5 concurrent tasks).
    *   Up to 10,000 API requests per month.
    *   Access to advanced agents and specialized tools (e.g., advanced code review, automated testing, marketing content generation).
    *   Priority support (email & chat, 12-hour response time).
    *   Basic analytics and reporting on agent usage.
    *   Team collaboration features (e.g., shared workspaces for tasks and agent configurations).
*   **Price:**
    *   **Monthly:** $99 USD
    *   **Annually:** $999 USD (approximately 16% discount)
*   **Billing Model:** Monthly or Annual subscription.

### 1.3. Enterprise Tier
*   **Target User Profile:** Large enterprises, organizations with complex AI integration needs, high-volume usage, and custom requirements.
*   **Key Features:**
    *   **All Pro Tier features.**
    *   High-volume concurrent agent runs (custom negotiated limits).
    *   High-volume API requests (custom negotiated limits or unlimited).
    *   Dedicated agents and custom tool development upon request.
    *   Dedicated account manager and 24/7 premium support with SLA.
    *   Advanced analytics, custom reports, and system health monitoring.
    *   On-premise or private cloud deployment options.
    *   Single Sign-On (SSO) integration.
    *   Custom contract terms.
*   **Price:** Custom pricing (Requires direct sales contact).
*   **Billing Model:** Annual contract, custom terms.

## 2. Agentic Software Development Services (Productized Services)

These are fixed-scope, agent-delivered development and audit packages, ideal for one-time needs or specific project tasks.

### 2.1. Micro-Feature Development Package
*   **Description:** Delivery of a small, well-defined software feature (e.g., integrate a new third-party API, add a specific form validation, create a simple utility script). Includes agent-driven code generation, automated testing, and deployment assistance to a staging environment.
*   **Deliverables:** Functional code, test reports, deployment instructions.
*   **Price:** $499 - $1,499 USD (fixed price, estimated based on complexity).

### 2.2. Website Audit & Optimization Report
*   **Description:** Automated, agent-powered comprehensive audit of a target website for performance bottlenecks, SEO compliance, potential security vulnerabilities, and code quality issues. Generates a detailed report with prioritized, actionable recommendations.
*   **Deliverables:** Comprehensive audit report (PDF/Markdown), prioritized recommendations.
*   **Price:** $299 - $799 USD (fixed price).

### 2.3. Automated API Endpoint Creation
*   **Description:** Agent-driven development and testing of a single RESTful API endpoint based on provided functional and technical specifications. Includes basic data model definition (if required) and implementation of standard CRUD (Create, Read, Update, Delete) operations.
*   **Deliverables:** Functional API endpoint code, automated test suite, API documentation.
*   **Price:** $799 - $1,999 USD (fixed price).
